 Resume ATS Screener

This project compares a resume and job description using TF-IDF and Cosine Similarity.

 Features:
- Calculates match percentage
- Removes stopwords
- Detects missing keywords
- Uses UTF-8 encoding

 Technologies Used:
- Python
- Scikit-learn
- NLP (TF-IDF)
- Cosine Similarity

 How to Run:

1. Install required libraries:
   pip install scikit-learn

2. Run the file
